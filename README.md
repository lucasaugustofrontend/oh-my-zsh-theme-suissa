## How to use

Copy the file `suissa.zsh-theme` to your oh-my-zsh theme folder `~/.oh-my-zsh/themes`.

Set `ZSH_THEME="suissa"` in your `~/.zshrc` file.

Restart your terminal.

Be happy.
